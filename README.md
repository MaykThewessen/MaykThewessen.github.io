# maykthewessen.github.io

Source for my personal site at <https://maykthewessen.github.io/>.

Single static page — `index.html` plus one CSS file. No build step. The
`.nojekyll` file tells GitHub Pages to serve files as-is instead of running
Jekyll.

## Editing

Edit `index.html` for content and `assets/css/style.css` for styling. Push
to `main` and GitHub Pages will redeploy in about a minute.

To preview locally:

```sh
python3 -m http.server 8000
# then open http://localhost:8000
```
